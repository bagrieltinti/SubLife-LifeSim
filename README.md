# SubLife
SubLife (LifeSim) - pt/br
Life Simulator
Simulador de vida baseado em InstLife/BitLife
https://discord.gg/YD6uVt3ddW
https://twitter.com/bagrieltintidev

## Melhorias recentes

- Sistema de empregos com múltiplas carreiras desbloqueadas a partir dos 18 anos, cada uma com salário anual e impacto na felicidade.
- Botão dedicado para gerenciar empregos direto da interface principal.
- Novas atividades (meditar e ensaiar com instrumentos) e efeitos mais completos que afetam felicidade e aparência.
